# php56-apache24-postfix

php5 系の開発におけるローカル環境の雛形です。

- php5.6
- Apache2.4
- MySQL8
- PostFix
- MailHog
  が含まれます。

# Usage

コンテナのビルド

```bash
docker compose build
```

コンテナの起動

```bash
docker compose up
```

- Web http://localhost:8000
- phpmyadmin http://localhost:8080
- MailHog http://localhost:8025
